## Скачать

Перейдите в раздел [**Releases**](https://github.com/k4t4my/task-tracker-releases/releases) и скачайте файл для вашей ОС:

| Платформа | Файл | Тип |
|-----------|------|-----|
| Windows | `Task.Tracker.*.exe` | Portable — запускается без установки |
| macOS | `Task.Tracker.*.dmg` | Образ диска — перетащите в Applications |
| Linux | `Task.Tracker.*.AppImage` | AppImage — сделайте исполняемым и запустите |

## Установка

### Windows

1. Скачайте `.exe` из последнего релиза
2. Поместите в удобную папку (например, `C:\Tools\`)
3. Запустите — установка не требуется

> Портативная версия: все данные хранятся в `%APPDATA%\task-tracker`, сам .exe можно переносить.

### macOS

1. Скачайте `.dmg` из последнего релиза
2. Откройте образ и перетащите приложение в папку Applications
3. При первом запуске: ПКМ → «Открыть» (обход Gatekeeper, приложение не подписано)

### Linux

1. Скачайте `.AppImage` из последнего релиза
2. Сделайте файл исполняемым:
   ```bash
   chmod +x Task.Tracker.*.AppImage
   ```
3. Запустите:
   ```bash
   ./Task.Tracker.*.AppImage
   ```

## Обновления

Приложение поддерживает автоматические обновления. При запуске проверяется наличие новой версии, и если обновление доступно — оно скачивается и устанавливается автоматически.

Также можно проверить обновления вручную: **Настройки → Обновления → Проверить обновления**.

### Каналы обновлений

- **Stable** — стабильные релизы, рекомендуется для повседневного использования
- **Beta** — предварительные версии с новым функционалом, возможны ошибки

Переключение каналов: **Настройки → Обновления → Канал обновлений**.

## Что это за приложение

Focus Tracker — десктопный тайм-трекер для работы с задачами. Основные возможности:

- **Таймер задач** — запуск, пауза, стоп с автоматической записью времени в Kanban
- **Помодоро** — 25/5 минут с звуковыми и визуальными оповещениями
- **Kanban-доска** — загрузка задач, фильтрация, переключение статусов
- **Календарь** — интеграция с Яндекс CalDAV, напоминания о созвонах за 5 минут
- **Jira** — быстрое создание задач по глобальному хоткею
- **Обед** — таймер с Telegram-уведомлениями в рабочие чаты
- **Заметки** — встроенный Markdown-блокнот
- **История** — список и timeline отработанных задач за день
- **Кастомизация** — светлая/тёмная тема, акцентный цвет, compact-режим

## Первоначальная настройка

При первом запуске:

1. Перейдите в **Настройки → Авторизация**
2. Нажмите **«Войти в Kanban»** и введите email/пароль от Kanban-доски
3. *(Опционально)* Укажите **CalDAV URL** календаря, логин и пароль приложения Яндекса
4. *(Опционально)* Укажите **логин/пароль Jira** и назначьте хоткей для быстрого создания задач
5. *(Опционально)* Добавьте **Telegram-чаты** для уведомлений об обеде
6. Нажмите **«Сохранить»**

## Хранение данных

Все данные хранятся локально на вашем компьютере:

- **Windows:** `%APPDATA%\task-tracker\`
- **macOS:** `~/Library/Application Support/task-tracker/`
- **Linux:** `~/.config/task-tracker/`

Файлы: `tasks.json` (история), `config.json` (настройки), `Notes/` (заметки).
