# Focus Tracker

Минималистичный тайм-трекер для Windows. Portable — установка не нужна, просто запусти `.exe`.

## Скачать

→ **[Последний релиз](../../releases/latest)**

Два канала обновлений:
- **Stable** (`v1.0`) — стабильные релизы
- **Beta** (`v1.0-beta.1`) — ранний доступ к новым функциям

## Возможности

- **Таймер задач** — запуск/пауза с сохранением истории по дням
- **Pomodoro** — встроенный таймер с настраиваемыми интервалами
- **Kanban** — загрузка и трекинг задач из внешней доски
- **Обед** — отправка уведомлений об уходе/возвращении через Telegram-бота
- **Компактный режим** — сворачивается до мини-виджета поверх всех окон
- **Календарь** — парсинг событий из Яндекс.Календаря
- **Тёмная/светлая тема**

## Установка

1. Скачай `Task Tracker *.exe` из [Releases](../../releases)
2. Запусти — установка не требуется
3. При первом запуске настрой аккаунт в разделе **Настройки**

Данные хранятся локально: `%APPDATA%\task-tracker\`

## Системные требования

- Windows 10 / 11 (x64)

## Исходный код

Приложение разрабатывается в закрытом репозитории.
